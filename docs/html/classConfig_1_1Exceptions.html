◆ $deprecationLogLevel

string Config\Exceptions::$deprecationLogLevel = LogLevel::WARNING

LOG LEVEL THRESHOLD FOR DEPRECATIONS

If $logDeprecations is set to true, this sets the log level to which the deprecation will be logged. This should be one of the log levels recognized by PSR-3.

The related Config\Logger::$threshold should be adjusted, if needed, to capture logging the deprecations.

Error Views Path
