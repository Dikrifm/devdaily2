Detailed Description

Image Processor Library

  • Infrastructure Layer. Handles low-level image manipulation using GD/Imagick via CI4 Services. Implements "Enterprise Sultan Mode":
  1. Thumb (150px)
  2. Medium (800px)
  3. Large (1920px, 95% Quality) All outputs are WebP.
